Settings

Operator tunables. Changes apply immediately and persist to gigot.json.

Push concurrency

Caps how many git-receive-pack requests run at the same time. When all slots are busy, additional pushes get 429 Too Many Requests with a Retry-After header so the client backs off and retries. Reads (clone / fetch) are not slot-gated.

Mirror remote-status polling

How often the server runs git ls-remote against each enabled mirror destination to refresh the in-sync / diverged badge. Toggle off to stop the background poller entirely; the per-destination Refresh button on Repositories still works. Successful pushes also mark the destination in-sync without an extra round-trip.
